BMI160: I2C <-> DMA (1 MHz)

Radio Controller: USART <-> DMA (115200)

Motors: DMA -> PWM 

Camera: DCMI -> DMA -> QSPI -> DMA -> SPI

Battery: ADC -> DMA

Temp comes from BMI160

Humedity: I2C <-> DMA

BME250: I2C <-> DMA (Humidity, Pressure, Temperature)

Battery, temperature, humedity, pressure -> each minute update

speed, displacement, height -> each second update
